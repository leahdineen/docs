## Overview

The `subtitles` detector identifies and analyzes subtitle and caption content in videos using a combination of OpenAI Vision API and OCR technology. It can verify subtitle presence and quality for accessibility compliance.

## Configuration

<ParamField body="config.require_subtitles" type="boolean">
  Whether subtitles are required for the content to pass evaluation
</ParamField>

## Example Configuration

```json
{
  "detector_name": "subtitles",
  "config": {
    "require_subtitles": true
  }
}
```

## Interpreting Results

The subtitles detector analyzes caption and subtitle quality and returns:

- **Subtitles Detected** - Boolean indicating whether subtitles are present in the video
- **Subtitle Overlays** - Array of detected subtitle text with timestamps, confidence scores, and readability assessments
- **Subtitle Coverage** - Percentage of video frames containing subtitles
- **Average Confidence** - Overall OCR confidence score for text detection accuracy
- **Readability Score** - Assessment of how easily subtitles can be read (0.0 to 1.0)
- **Quality Assessment** - Evaluation of text clarity, positioning, timing synchronization, and completeness
