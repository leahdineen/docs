## Overview

Detects the primary language and validates it against an allowed list.

## Configuration

<ParamField body="config.accepted_languages" type="array" required>
  List of accepted language codes (for example: "en", "es").
</ParamField>

## Example Configuration

```json
{
  "detector_name": "language",
  "config": {
    "accepted_languages": ["en", "es"]
  }
}
```

## Result Schema

```json
{
  "detector_name": "language",
  "pass_check": true,
  "score": 0.94,
  "rationale": "Detected language 'en' (confidence: 94.0%) is in accepted languages: ['en', 'es']",
  "metrics": {
    "detected_language": "en",
    "confidence": 0.94,
    "accepted_languages": ["en", "es"],
    "is_accepted_language": true,
    "passes_confidence_threshold": true,
    "confidence_threshold": 0.7,
    "all_detected_languages": [
      {"language": "en", "confidence": 0.94},
      {"language": "fr", "confidence": 0.03}
    ],
    "transcript_length": 824,
    "transcript_word_count": 142
  },
  "timestamps": [],
  "flagged_frames": [],
  "execution_time_sec": null,
  "api_cost_usd": null
}
```

## Interpreting Results

- **detected_language/confidence**: Primary language and confidence score.
- **passes_confidence_threshold**: Whether confidence meets the minimum threshold.
- **all_detected_languages**: Alternate language candidates with scores.
