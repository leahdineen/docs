## Overview

The `image_specs` detector validates image technical specifications including dimensions, aspect ratios, resolution requirements, and quality metrics. It ensures images meet specified technical standards for various use cases.

## Configuration

<ParamField body="config.allowed_aspect_ratios" type="array">
  List of allowed aspect ratios (e.g., ["16:9", "9:16", "1:1", "4:3"])
</ParamField>

<ParamField body="config.min_width" type="integer">
  Minimum image width in pixels
</ParamField>

<ParamField body="config.min_height" type="integer">
  Minimum image height in pixels
</ParamField>

<ParamField body="config.min_megapixels" type="number">
  Minimum image size in megapixels (width × height ÷ 1,000,000)
</ParamField>

## Example Configuration

```json
{
  "detector_name": "image_specs",
  "config": {
    "allowed_aspect_ratios": ["16:9", "1:1"],
    "min_width": 1080,
    "min_height": 1080,
    "min_megapixels": 2.0
  }
}
```

## Specification Categories

### Dimension Requirements
- **Width and Height** - Minimum pixel dimensions
- **Aspect Ratio** - Proportional relationship validation
- **Orientation** - Landscape, portrait, or square format
- **Standard Formats** - Common resolutions (HD, Full HD, 4K)

### Resolution Standards
- **Megapixel Calculation** - Total image resolution
- **Pixel Density** - Detail and clarity assessment
- **Print Resolution** - DPI/PPI considerations
- **Display Resolution** - Screen compatibility

### Quality Specifications
- **Color Depth** - Bit depth per channel
- **Color Space** - sRGB, Adobe RGB, etc.
- **Compression** - JPEG quality levels
- **File Format** - JPEG, PNG, WEBP support

## Aspect Ratio Validation

### Standard Ratios
- **16:9** - Widescreen (1920×1080, 1280×720)
- **9:16** - Vertical/mobile (1080×1920, 720×1280)
- **1:1** - Square (1080×1080, 1000×1000)
- **4:3** - Traditional (1024×768, 800×600)
- **3:2** - Photography (1500×1000, 3000×2000)
- **21:9** - Ultra-wide (2560×1080, 3440×1440)

### Custom Ratios
- **Golden Ratio** - 1.618:1 (φ ratio)
- **Cinema Formats** - 2.39:1, 1.85:1
- **Social Media** - Platform-specific ratios
- **Print Formats** - A4, Letter, etc.

## Quality Assessment

### Image Quality Metrics
- **Color Accuracy** - Faithful color reproduction
- **Dynamic Range** - Tonal range and contrast
- **Noise Levels** - Digital noise assessment
- **Sharpness** - Edge definition and clarity

### File Analysis
- **Compression Quality** - Lossy vs. lossless assessment
- **File Size** - Optimization and efficiency
- **Metadata** - EXIF and color profile information
- **Format Suitability** - Best format for use case

## Error Conditions

### Dimension Failures
- **Too Small** - Below minimum width/height
- **Wrong Ratio** - Incorrect aspect ratio
- **Insufficient Resolution** - Below megapixel threshold
- **Orientation Mismatch** - Landscape vs. portrait issues

### Quality Issues
- **Poor Compression** - Over-compressed images
- **Wrong Format** - Inappropriate file type
- **Color Space** - Incorrect color profile
- **Missing Metadata** - Required information absent

## Detector Type

**HEURISTIC** - Uses image processing algorithms for specification analysis and validation
