## Overview

The `transition_detection` detector identifies scene changes, cuts, and transitions in video content using multiple OpenCV-based detection methods including histogram correlation, edge detection difference, and pixel difference analysis.

## Configuration

<ParamField body="config.max_transitions" type="integer">
  Maximum number of allowed scene transitions before failing the evaluation
</ParamField>

## Example Configuration

```json
{
  "detector_name": "transition_detection",
  "config": {
    "max_transitions": 10
  }
}
```

## Interpreting Results

The transition_detection detector analyzes video cuts and scene changes and returns:

- **Total Transitions** - Number of scene transitions detected throughout the video
- **Detected Transitions** - Array of transitions with timestamps, types, and confidence scores
- **Transition Type** - Classification of transition (cut, fade, dissolve, jump cut, etc.)
- **Detection Method** - Which analysis method identified the transition (histogram, edge detection, pixel difference)
- **Intensity** - Strength of the transition (strong, moderate, weak, minimal)
- **Scene Segments** - Individual scenes between transitions with duration analysis
