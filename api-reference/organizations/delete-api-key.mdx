---
title: 'Delete API Key'
api: 'DELETE /api-keys/{key_id}'
description: 'Delete an organization API key'
---

## Overview

Deletes an API key, immediately revoking access for any applications using it. This action is irreversible.

## Path Parameters

<ParamField path="key_id" type="string" required>
  Unique identifier of the API key to delete
</ParamField>

## Response

<ResponseField name="deleted" type="boolean" required>
  Whether the key was successfully deleted
</ResponseField>

<ResponseField name="api_key_id" type="string" required>
  ID of the deleted key
</ResponseField>

<ResponseField name="deleted_at" type="datetime" required>
  Deletion timestamp
</ResponseField>

## Examples

<CodeGroup>

```bash Request
curl -X DELETE https://api.judge.example.com/api-keys/key_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Success Response (200)
{
  "deleted": true,
  "api_key_id": "key_abc123",
  "deleted_at": "2024-01-15T16:00:00Z"
}
```

</CodeGroup>
