---
title: 'Delete Judge (API Key)'
api: 'DELETE /api-key/judges/{judge_id}'
description: 'Delete a judge using API key authentication'
---

## Overview

Deletes a judge using API key authentication. This action is irreversible and will remove all associated data.

## Headers

<ParamField header="X-API-Key" type="string" required>
  Organization API key for authentication
</ParamField>

## Path Parameters

<ParamField path="judge_id" type="string" required>
  Unique identifier of the judge to delete
</ParamField>

## Response

<ResponseField name="deleted" type="boolean" required>
  Whether the judge was successfully deleted
</ResponseField>

<ResponseField name="judge_id" type="string" required>
  ID of the deleted judge
</ResponseField>

<ResponseField name="deleted_at" type="datetime" required>
  Deletion timestamp
</ResponseField>

## Examples

<CodeGroup>

```bash Request
curl -X DELETE https://api.getpique.ai/api-key/judges/jdg_xyz789 \
  -H "X-API-Key: sk_org_xxxxxxxxx"
```

```json Success Response (200)
{
  "deleted": true,
  "judge_id": "jdg_xyz789",
  "deleted_at": "2024-01-15T16:45:00Z"
}
```

</CodeGroup>
