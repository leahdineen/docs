---
title: 'Delete API Key Evaluation'
api: 'DELETE /evaluations/{job_id}'
description: 'Delete evaluation job and results using API key'
---

## Overview

Deletes an evaluation job and all associated results. This action is irreversible and will permanently remove all evaluation data for the specified job. Jobs that are currently processing can be cancelled and deleted.

## Headers

<ParamField header="X-API-Key" type="string" required>
  Organization API key for authentication
</ParamField>

## Path Parameters

<ParamField path="job_id" type="string" required>
  Unique identifier of the evaluation job to delete
</ParamField>

## Query Parameters

<ParamField query="force" type="boolean" default="false">
  Force deletion of processing jobs (cancels and deletes)
</ParamField>

## Response

<ResponseField name="job_id" type="string" required>
  ID of the deleted job
</ResponseField>

<ResponseField name="status" type="string" required>
  Final job status before deletion
</ResponseField>

<ResponseField name="deleted_at" type="datetime" required>
  Timestamp when the job was deleted
</ResponseField>

<ResponseField name="items_deleted" type="integer" required>
  Number of evaluation results deleted
</ResponseField>

<ResponseField name="was_cancelled" type="boolean" required>
  Whether the job was cancelled during deletion
</ResponseField>

<ResponseField name="message" type="string" required>
  Confirmation message
</ResponseField>

## Examples

<CodeGroup>

```bash Request: Delete Completed Job
curl -X DELETE https://api.judge.example.com/evaluations/job_batch_456 \
  -H "X-API-Key: YOUR_API_KEY"
```

```json Success Response (200)
{
  "job_id": "job_batch_456",
  "status": "completed",
  "deleted_at": "2024-01-15T11:00:00Z",
  "items_deleted": 25,
  "was_cancelled": false,
  "message": "Evaluation job and all associated results have been permanently deleted"
}
```

</CodeGroup>

<CodeGroup>

```bash Request: Force Delete Processing Job
curl -X DELETE "https://api.judge.example.com/evaluations/job_processing_789?force=true" \
  -H "X-API-Key: YOUR_API_KEY"
```

```json Force Delete Response (200)
{
  "job_id": "job_processing_789",
  "status": "processing",
  "deleted_at": "2024-01-15T11:01:00Z",
  "items_deleted": 15,
  "was_cancelled": true,
  "message": "Processing job was cancelled and deleted along with partial results"
}
```

</CodeGroup>

<CodeGroup>

```bash Error: Job Not Found
curl -X DELETE https://api.judge.example.com/evaluations/invalid_job_id \
  -H "X-API-Key: YOUR_API_KEY"
```

```json Error Response (404)
{
  "error": {
    "code": "JOB_NOT_FOUND",
    "message": "Evaluation job with ID 'invalid_job_id' not found or access denied"
  }
}
```

</CodeGroup>

<CodeGroup>

```bash Error: Cannot Delete Processing Job
curl -X DELETE https://api.judge.example.com/evaluations/job_processing_789 \
  -H "X-API-Key: YOUR_API_KEY"
```

```json Error Response (409)
{
  "error": {
    "code": "JOB_IN_PROGRESS",
    "message": "Cannot delete job that is currently processing. Use force=true to cancel and delete, or wait for completion",
    "current_status": "processing",
    "suggestion": "Add ?force=true to cancel and delete the job"
  }
}
```

</CodeGroup>

<CodeGroup>

```bash Error: Insufficient Permissions
curl -X DELETE https://api.judge.example.com/evaluations/job_other_org_123 \
  -H "X-API-Key: YOUR_API_KEY"
```

```json Error Response (403)
{
  "error": {
    "code": "ACCESS_DENIED",
    "message": "Insufficient permissions to delete this evaluation job"
  }
}
```

</CodeGroup>
