title: 'Create API Key'
api: 'POST /api-keys'
description: 'Create a new API key for organization'
---

## Overview

Creates a new API key for programmatic access to the Judge API. API keys provide secure access without requiring user authentication and can be configured with specific permissions and usage limits.

## Request Body

<ParamField body="name" type="string" required maxLength="100">
  Descriptive name for the API key
</ParamField>

<ParamField body="description" type="string" maxLength="500">
  Optional description of the API key's intended use
</ParamField>

<ParamField body="permissions" type="array" required>
  List of permissions for this API key
  - `evaluations.create` - Create evaluations
  - `evaluations.read` - Read evaluation results
  - `evaluations.delete` - Delete evaluations
  - `judges.read` - Read judge information
  - `content.upload` - Upload content
  - `content.read` - Read content
</ParamField>

<ParamField body="rate_limits" type="object">
  Rate limiting configuration
  
  <Expandable title="rate limit properties">
    <ParamField body="requests_per_minute" type="integer" default="100" min="10" max="1000">
      Maximum requests per minute
    </ParamField>
    
    <ParamField body="evaluations_per_hour" type="integer" default="500" min="50" max="5000">
      Maximum evaluations per hour
    </ParamField>
    
    <ParamField body="monthly_limit" type="integer" default="50000">
      Monthly evaluation limit
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="expires_at" type="datetime">
  Optional expiration date for the API key (ISO 8601)
</ParamField>

## Response

<ResponseField name="api_key_id" type="string" required>
  Unique identifier for the API key
</ResponseField>

<ResponseField name="name" type="string" required>
  API key name
</ResponseField>

<ResponseField name="api_key" type="string" required>
  The actual API key (only shown once)
</ResponseField>

<ResponseField name="permissions" type="array" required>
  Configured permissions
</ResponseField>

<ResponseField name="rate_limits" type="object" required>
  Configured rate limits
</ResponseField>

<ResponseField name="status" type="string" required>
  API key status (active, suspended, expired)
</ResponseField>

<ResponseField name="created_at" type="datetime" required>
  Creation timestamp
</ResponseField>

<ResponseField name="expires_at" type="datetime">
  Expiration timestamp (if set)
</ResponseField>

## Examples

```bash Request

  -H "X-API-Key: sk_org_xxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Production Evaluation API",
    "description": "API key for production content evaluation pipeline",
    "permissions": [
      "evaluations.create",
      "evaluations.read",
      "judges.read",
      "content.upload",
      "content.read"
    ],
    "rate_limits": {
      "requests_per_minute": 200,
      "evaluations_per_hour": 1000,
      "monthly_limit": 100000
    },
    "expires_at": "2025-01-15T00:00:00Z"
  }'
```
