## Overview

The `transcript_product` detector analyzes video transcripts to detect mentions of specific brand and product names. It uses keyword-based matching to identify when products are mentioned and discussed in the content.

**Requires at least one configuration parameter** - Either `brand_name` or `product_name` must be provided.

## Configuration

<ParamField body="config.brand_name" type="string">
  Brand name to detect in the transcript
</ParamField>

<ParamField body="config.product_name" type="string">
  Product name to detect in the transcript
</ParamField>

## Example Configuration

```json
{
  "detector_name": "transcript_product",
  "config": {
    "brand_name": "Pique",
    "product_name": "SuperClean Face Wash"
  }
}
```

## Interpreting Results

The transcript_product detector analyzes transcript content for brand and product mentions and returns:

- **Brand Mentions** - Array of detected brand name references with timestamps, confidence, and context
- **Product Mentions** - Array of detected product name references with usage context and sentiment
- **Mention Summary** - Total counts, frequency analysis, first mention timing, and overall sentiment
- **Context Analysis** - Categorization of mention types (endorsement, testimonial, review, disclosure, etc.)
- **Confidence Scores** - Detection confidence from 0.0 to 1.0 for each mention identified
