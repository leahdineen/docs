---
title: 'Get Content Notes'
api: 'GET /content/{content_id}/notes'
description: 'Retrieve notes for content item'
---

## Overview

Retrieves all notes and annotations for a content item.

## Path Parameters

<ParamField path="content_id" type="string" required>
  The unique identifier of the content
</ParamField>

## Response

<ResponseField name="notes" type="array" required>
  Array of note objects
  
  <Expandable title="note properties">
    <ResponseField name="note_id" type="string" required>
      Note identifier
    </ResponseField>
    
    <ResponseField name="note" type="string" required>
      Note content
    </ResponseField>
    
    <ResponseField name="category" type="string">
      Note category
    </ResponseField>
    
    <ResponseField name="created_at" type="datetime" required>
      Creation timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="integer" required>
  Total number of notes
</ResponseField>

## Examples

<CodeGroup>

```bash Request
curl -X GET https://api.getpique.ai/content/cnt_abc123/notes \
  -H "X-API-Key: sk_org_xxxxxxxxx"
```
