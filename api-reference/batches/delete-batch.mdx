---
title: 'Delete Batch'
api: 'DELETE /batches/{batch_id}'
description: 'Delete a batch job'
---

## Overview

Deletes a batch job and all associated evaluation results. Active evaluations will be cancelled.

<Warning>
This will permanently delete the batch and all its evaluation results. This action cannot be undone.
</Warning>

## Path Parameters

<ParamField path="batch_id" type="string" required>
  The unique identifier of the batch to delete
</ParamField>

## Response

Returns `204 No Content` on successful deletion.

## Examples

<CodeGroup>

```bash Request
curl -X DELETE https://api.judge.example.com/batches/batch_def456 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash Success Response (204)
# No response body - successful deletion
```

</CodeGroup>

<CodeGroup>

```bash Error: Batch Not Found
curl -X DELETE https://api.judge.example.com/batches/invalid_id \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Error Response (404)
{
  "error": {
    "code": "BATCH_NOT_FOUND",
    "message": "Batch with ID 'invalid_id' not found"
  }
}
```

</CodeGroup>

<CodeGroup>

```bash Error: Batch In Progress
curl -X DELETE https://api.judge.example.com/batches/batch_active \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```json Error Response (400)
{
  "error": {
    "code": "BATCH_IN_PROGRESS",
    "message": "Cannot delete batch while evaluations are in progress. Cancel the batch first."
  }
}
```

</CodeGroup>
