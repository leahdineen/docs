Each detector analyzes specific aspects of content and can be configured with custom parameters. Click on any detector below for detailed configuration options and result schemas.

## Content Matching Detectors

| Detector | Description |
|----------|-------------|
| [audio_matching](/api-reference/detectors/audio-matching) | Checks whether audio matches reference audio |
| [splitscreen_video_matching](/api-reference/detectors/splitscreen-video-matching) | Checks split-screen content against references or descriptions |
| [video_matching](/api-reference/detectors/video-matching) | Checks whether video matches reference videos |
| [video_matching_description](/api-reference/detectors/video-matching-description) | Checks whether video content matches a description |
