## Overview

The `background` detector analyzes the background and setting of video content to assess environment quality, cleanliness, and appropriateness.

**No configuration required** - This detector works with default settings.

## Configuration

This detector does not require any configuration parameters. Include it with an empty config object:

```json
{
  "detector_name": "background",
  "config": {}
}
```

## Example Configuration

```json
{
  "detector_name": "background",
  "config": {}
}
```

## Interpreting Results

- **is_car**: Whether the background is inside a vehicle
- **is_tidy**: Boolean indicating if the background is clean and organized
- **environment_type**: Category of background setting (indoor_studio, home_setting, outdoor, etc.)
- **professionalism**: Overall professionalism level (high, medium, low, poor)
- **visual_appeal**: Score from 0.0-1.0 rating the background's visual quality
- **distraction_level**: Score from 0.0-1.0 indicating how distracting the background is
