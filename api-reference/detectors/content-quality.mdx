## Overview

The `content_quality` detector provides a comprehensive assessment of overall video content quality, analyzing multiple factors including visual appeal, production value, engagement potential, and overall professionalism.

**No configuration required** - This detector works with default settings.

## Configuration

This detector does not require any configuration parameters. Include it with an empty config object:

```json
{
  "detector_name": "content_quality",
  "config": {}
}
```

## Example Configuration

```json
{
  "detector_name": "content_quality",
  "config": {}
}
```

## Interpreting Results

- **production_value**: Score from 0.0-1.0 rating technical execution and setup quality
- **visual_appeal**: Score from 0.0-1.0 rating aesthetic quality and composition
- **content_engagement**: Score from 0.0-1.0 rating pacing, energy, and authenticity
- **professionalism**: Score from 0.0-1.0 rating overall professional presentation
- **quality_factors**: Detailed breakdown of lighting, composition, audio sync, and stability
- **improvement_areas**: Suggestions for enhancing content quality
