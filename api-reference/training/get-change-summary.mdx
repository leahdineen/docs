---
title: 'Get Change Summary'
api: 'GET /jobs/{job_id}/change-summary'
description: 'Get summary of changes made during training'
---

## Overview

Retrieves a summary of changes and improvements made during a training job, including performance deltas and specific modifications.

## Path Parameters

<ParamField path="job_id" type="string" required>
  Unique identifier of the training job
</ParamField>

## Response

<ResponseField name="job_id" type="string" required>
  Training job ID
</ResponseField>

<ResponseField name="changes_summary" type="object" required>
  Summary of changes made
  
  <Expandable title="changes properties">
    <ResponseField name="criteria_modifications" type="array" required>
      Changes made to evaluation criteria
    </ResponseField>
    
    <ResponseField name="performance_improvements" type="object" required>
      Performance improvements achieved
    </ResponseField>
    
    <ResponseField name="model_updates" type="object" required>
      Updates to the underlying model
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="before_after_comparison" type="object" required>
  Before and after performance comparison
</ResponseField>

<ResponseField name="generated_at" type="datetime" required>
  Summary generation timestamp
</ResponseField>

## Examples
