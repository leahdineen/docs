## Overview

The `image_ip_infringement` detector analyzes visual content to identify potential intellectual property violations, including unauthorized use of copyrighted material, trademarked content, or brand elements that may infringe on IP rights.

## Configuration

<ParamField body="config.allowed_terms" type="array">
  List of allowed trademarked terms, brands, or IP elements that are permitted in the content
</ParamField>

## Example Configuration

```json
{
  "detector_name": "image_ip_infringement",
  "config": {
    "allowed_terms": [
      "Nike",
      "Apple", 
      "Microsoft",
      "approved_brand_logos"
    ]
  }
}
```

## Interpreting Results

The image_ip_infringement detector identifies potential intellectual property violations and returns:

- **IP Violations** - Array of detected violations including trademark, copyright, or design patent issues
- **Violation Type** - Category of violation (trademark, copyrighted_content, brand_logos, etc.)
- **Severity Level** - High, medium, or low severity based on legal risk assessment  
- **Confidence Score** - Detection confidence from 0.0 to 1.0
- **Risk Level** - Overall IP risk assessment (critical, high, medium, low)
- **Allowed Elements** - IP elements that are permitted based on configuration
