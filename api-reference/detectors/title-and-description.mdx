## Overview

The `title_and_description` detector analyzes video metadata including title, description, hashtags, and social media tags for compliance and optimization. It checks for required elements and analyzes content topics.

**Requires at least one configuration parameter** - At least one configuration option must be provided to make the analysis meaningful.

## Configuration

<ParamField body="config.suggested_hashtags" type="array">
  List of suggested hashtags to look for (optional recommendations)
</ParamField>

<ParamField body="config.required_hashtags" type="array">
  List of hashtags that must be present
</ParamField>

<ParamField body="config.required_social_tags" type="array">
  List of social media tags that must be present (e.g., @mentions)
</ParamField>

<ParamField body="config.required_social_tags_mode" type="string">
  Mode for social tag requirements: "all" (all tags required) or "any" (at least one required)
</ParamField>

<ParamField body="config.topic_keywords" type="array">
  List of topic keywords to detect in title/description
</ParamField>

<ParamField body="config.enable_topic_analysis" type="boolean">
  Whether to enable AI-powered topic analysis of the content
</ParamField>

## Example Configuration

```json
{
  "detector_name": "title_and_description",
  "config": {
    "suggested_hashtags": ["#beauty", "#skincare", "#tutorial"],
    "required_hashtags": ["#ad", "#sponsored"],
    "required_social_tags": ["@brandname"],
    "required_social_tags_mode": "any",
    "topic_keywords": ["skincare", "routine", "product"],
    "enable_topic_analysis": true
  }
}
```

## Interpreting Results

The title_and_description detector analyzes video metadata for compliance and optimization, returning:

- **Hashtag Analysis** - Coverage of suggested and required hashtags with missing tags identified
- **Social Tag Analysis** - Verification of required @mentions and social media attribution
- **Topic Analysis** - AI-powered detection of content topics and keyword matches
- **Topic Relevance** - Score from 0.0 to 1.0 indicating content alignment with expected topics
- **Compliance Score** - Overall adherence to requirements including disclosure and attribution standards
- **Metadata Summary** - Title analysis, description length, hashtag count, and mention count
