## Overview

The `pronunciation` detector analyzes audio to detect and evaluate pronunciation quality of brand and product names. It uses Google Gemini AI to assess pronunciation correctness, clarity, and confidence scores.

**Requires at least one configuration parameter** - Either `brand_name` or `product_name` must be provided.

## Configuration

<ParamField body="config.brand_name" type="string">
  Brand name to check pronunciation for
</ParamField>

<ParamField body="config.product_name" type="string">
  Product name to check pronunciation for
</ParamField>

<ParamField body="config.brand_cue" type="string">
  Phonetic cue or pronunciation guide for the brand name
</ParamField>

<ParamField body="config.product_cue" type="string">
  Phonetic cue or pronunciation guide for the product name
</ParamField>

<ParamField body="config.brand_audio_url" type="string">
  GCS URI of reference audio for brand name pronunciation
  
  **Allowed extensions:** mp3, wav, aac, flac, m4a
</ParamField>

<ParamField body="config.product_audio_url" type="string">
  GCS URI of reference audio for product name pronunciation
  
  **Allowed extensions:** mp3, wav, aac, flac, m4a
</ParamField>

## Example Configuration

```json
{
  "detector_name": "pronunciation",
  "config": {
    "brand_name": "Pique",
    "brand_cue": "pronounced like 'peak'",
    "product_name": "SuperClean",
    "brand_audio_url": "gs://judge-artifacts/org_123/reference_uploads/abc/brand_audio.mp3"
  }
}
```
