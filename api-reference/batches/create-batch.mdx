---
title: 'Create Batch'
api: 'POST /batches'
description: 'Create a batch evaluation job'
---

## Overview

Creates a batch job to evaluate multiple content items against one or more judges efficiently.

## Request Body

<ParamField body="name" type="string" required maxLength="200">
  Name for the batch job
</ParamField>

<ParamField body="description" type="string" maxLength="1000">
  Description of the batch job
</ParamField>

<ParamField body="judge_ids" type="array" required minLength="1">
  Array of judge IDs to evaluate content against
</ParamField>

<ParamField body="content_ids" type="array" required minLength="1">
  Array of content IDs to evaluate
</ParamField>

<ParamField body="priority" type="string" default="normal">
  Batch priority level
  - `low` - Lower priority processing
  - `normal` - Standard processing  
  - `high` - Priority processing
</ParamField>

## Response

<ResponseField name="batch_id" type="string" required>
  Unique identifier for the batch
</ResponseField>

<ResponseField name="name" type="string" required>
  Batch name
</ResponseField>

<ResponseField name="status" type="string" required>
  Batch status
  - `created` - Batch created, not started
  - `evaluating` - Evaluations in progress
  - `completed` - All evaluations finished
  - `failed` - Batch failed
</ResponseField>

<ResponseField name="total_evaluations" type="integer" required>
  Total number of evaluations to be performed
</ResponseField>

<ResponseField name="completed_evaluations" type="integer" required>
  Number of completed evaluations
</ResponseField>

<ResponseField name="created_at" type="datetime" required>
  Creation timestamp
</ResponseField>

## Examples

<CodeGroup>

```bash Request
curl -X POST https://api.judge.example.com/batches \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 Content Review",
    "description": "Quarterly review of all blog content",
    "judge_ids": ["jdg_xyz789", "jdg_abc123"],
    "content_ids": ["cnt_001", "cnt_002", "cnt_003"],
    "priority": "normal"
  }'
```

```json Success Response (201)
{
  "batch_id": "batch_def456",
  "name": "Q1 Content Review",
  "status": "created",
  "total_evaluations": 6,
  "completed_evaluations": 0,
  "created_at": "2024-01-15T10:00:00Z"
}
```

</CodeGroup>

<CodeGroup>

```bash Error: Invalid Judge
curl -X POST https://api.judge.example.com/batches \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Test Batch",
    "judge_ids": ["invalid_judge"],
    "content_ids": ["cnt_001"]
  }'
```

```json Error Response (400)
{
  "error": {
    "code": "INVALID_JUDGE",
    "message": "Judge with ID 'invalid_judge' not found"
  }
}
```

</CodeGroup>
