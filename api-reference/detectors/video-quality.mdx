## Overview

The `video_quality` detector provides comprehensive analysis of video technical quality, including compression artifacts, clarity, visual noise, and overall production standards. It evaluates multiple technical aspects that affect viewing experience.

## Configuration

<ParamField body="config.expected_aspect_ratios" type="array">
  Expected aspect ratios for the content (e.g., ["16:9", "9:16"])
</ParamField>

<ParamField body="config.check_aspect_ratio" type="boolean">
  Whether to validate aspect ratio consistency throughout the video
</ParamField>

<ParamField body="config.min_width" type="integer">
  Minimum acceptable video width in pixels
</ParamField>

<ParamField body="config.min_height" type="integer">
  Minimum acceptable video height in pixels
</ParamField>

## Example Configuration

```json
{
  "detector_name": "video_quality",
  "config": {
    "expected_aspect_ratios": ["16:9", "9:16"],
    "check_aspect_ratio": true,
    "min_width": 1280,
    "min_height": 720
  }
}
```

## Interpreting Results

The video_quality detector analyzes technical video quality and returns:

- **Technical Quality** - Comprehensive analysis including resolution, compression, and visual metrics
- **Compression Analysis** - Assessment of encoding quality, bitrate estimation, and artifact detection
- **Visual Metrics** - Scores for sharpness, contrast, color accuracy, and noise levels (0.0 to 1.0)
- **Quality Issues** - Array of specific problems detected with timestamps, severity, and descriptions
- **Encoding Assessment** - Evaluation of codec efficiency, bitrate consistency, and detail preservation
- **Frame Analysis** - Statistics on problematic frames and overall quality consistency
