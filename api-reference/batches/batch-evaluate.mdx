---
title: 'Batch Evaluate Content'
api: 'POST /batches/evaluate'
description: 'Create and start a batch evaluation job'
---

## Overview

Creates and immediately starts a batch evaluation job. This is a convenience endpoint that combines batch creation with immediate execution.

## Request Body

<ParamField body="name" type="string" required maxLength="200">
  Name for the batch job
</ParamField>

<ParamField body="description" type="string" maxLength="1000">
  Description of the batch job
</ParamField>

<ParamField body="judge_ids" type="array" required minLength="1">
  Array of judge IDs to evaluate content against
</ParamField>

<ParamField body="content_items" type="array" required minLength="1">
  Array of content items to evaluate
  
  <Expandable title="content item properties">
    <ParamField body="content_id" type="string" required>
      ID of existing content, if using uploaded content
    </ParamField>
    
    <ParamField body="text_content" type="string">
      Direct text content to evaluate
    </ParamField>
    
    <ParamField body="name" type="string" required>
      Name for this content item
    </ParamField>
    
    <ParamField body="metadata" type="object">
      Additional metadata for the content
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="priority" type="string" default="normal">
  Batch priority (low, normal, high)
</ParamField>

<ParamField body="auto_start" type="boolean" default="true">
  Whether to start evaluation immediately
</ParamField>

## Response

<ResponseField name="batch_id" type="string" required>
  Unique identifier for the batch
</ResponseField>

<ResponseField name="name" type="string" required>
  Batch name
</ResponseField>

<ResponseField name="status" type="string" required>
  Batch status (created, evaluating)
</ResponseField>

<ResponseField name="total_evaluations" type="integer" required>
  Total evaluations to perform
</ResponseField>

<ResponseField name="estimated_completion_time" type="datetime">
  Estimated completion time
</ResponseField>

<ResponseField name="created_at" type="datetime" required>
  Creation timestamp
</ResponseField>

## Examples

<CodeGroup>

```bash Request
curl -X POST https://api.judge.example.com/batches/evaluate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Daily Content Review",
    "description": "Review all today'\''s content",
    "judge_ids": ["jdg_xyz789"],
    "content_items": [
      {
        "content_id": "cnt_001"
      },
      {
        "text_content": "This is inline content to evaluate",
        "name": "Inline Content Test",
        "metadata": {"source": "api"}
      }
    ],
    "priority": "high",
    "auto_start": true
  }'
```

```json Success Response (201)
  "status": "evaluating",

```
