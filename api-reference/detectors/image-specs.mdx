## Overview

The `image_specs` detector validates image technical specifications including dimensions, aspect ratios, resolution requirements, and quality metrics. It ensures images meet specified technical standards for various use cases.

## Configuration

<ParamField body="config.allowed_aspect_ratios" type="array">
  List of allowed aspect ratios (e.g., ["16:9", "9:16", "1:1", "4:3"])
</ParamField>

<ParamField body="config.min_width" type="integer">
  Minimum image width in pixels
</ParamField>

<ParamField body="config.min_height" type="integer">
  Minimum image height in pixels
</ParamField>

<ParamField body="config.min_megapixels" type="number">

## Interpreting Results

The image_specs detector validates technical specifications and returns:

- **Image Specifications** - Actual dimensions, aspect ratio, megapixels, and color properties
- **Compliance Check** - Boolean flags indicating whether each requirement is met
- **Quality Metrics** - Pixel density, color accuracy, dynamic range, and file optimization metrics
- **Technical Analysis** - Format details, compression quality, metadata presence, and bit depth
- **Dimension Analysis** - Total pixels, orientation classification, and standard format identification
