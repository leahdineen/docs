---
title: 'Start Training Job'
api: 'POST /jobs/{job_id}/start'
description: 'Start execution of a training job'
---

## Overview

Starts execution of a created training job. This endpoint triggers the actual training process for judge improvement.

## Path Parameters

<ParamField path="job_id" type="string" required>
  Unique identifier of the training job to start
</ParamField>

## Response

<ResponseField name="job_id" type="string" required>
  ID of the started training job
</ResponseField>

<ResponseField name="status" type="string" required>
  Updated job status (started, queued, processing)
</ResponseField>

<ResponseField name="started_at" type="datetime" required>
  Job start timestamp
</ResponseField>

<ResponseField name="estimated_completion" type="datetime">
  Estimated completion time
</ResponseField>

## Examples

<CodeGroup>

```bash Request
curl -X POST https://api.getpique.ai/jobs/train_job_123/start \
  -H "X-API-Key: sk_org_xxxxxxxxx"
```

```json Success Response (202)
{
  "job_id": "train_job_123",
  "status": "processing",
  "started_at": "2024-01-15T17:45:00Z",
  "estimated_completion": "2024-01-15T18:30:00Z"
}
```

</CodeGroup>
