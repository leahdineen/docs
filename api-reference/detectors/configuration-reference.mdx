Each detector analyzes specific aspects of content and can be configured with custom parameters. Click on any detector below for detailed configuration options and result schemas.

## Content Matching Detectors

| Detector | Description |
|----------|-------------|
| [audio_matching](/api-reference/detectors/audio-matching) | Matches audio against reference content using chromaprint fingerprinting |
| [splitscreen_video_matching](/api-reference/detectors/splitscreen-video-matching) | Matches split-screen video content using multi-embedder alignment |
| [video_matching](/api-reference/detectors/video-matching) | Detects source video clip patterns using reference features |
| [video_matching_description](/api-reference/detectors/video-matching-description) | Matches video against text description |
