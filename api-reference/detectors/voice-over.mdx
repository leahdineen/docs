## Overview

The `voice_over` detector identifies and analyzes voice-over narration in video content. It can detect whether voice-over is present and optionally fail content based on voice-over detection rules.

## Configuration

<ParamField body="config.fail_if_voice_over" type="boolean">
  Whether to fail the evaluation if voice-over is detected
</ParamField>

## Example Configuration

```json
{
  "detector_name": "voice_over",
  "config": {
    "fail_if_voice_over": true
  }
}
```

## Interpreting Results

- **voice_over_detected**: Whether voice-over narration was found in the content
- **voice_over_percentage**: Percentage of total video duration containing voice-over
- **confidence**: Detection confidence score (0.0-1.0)
- **characteristics**: Voice type, speech rate, and audio quality metrics
