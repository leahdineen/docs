---
title: 'Get Feedback Summary'
api: 'GET /jobs/{job_id}/feedback-summary'
description: 'Get feedback summary for a training job'
---

## Overview

Retrieves a summary of feedback data used in a training job, including statistics and patterns that influenced the training.

## Path Parameters

<ParamField path="job_id" type="string" required>
  Unique identifier of the training job
</ParamField>

## Response

<ResponseField name="job_id" type="string" required>
  Training job ID
</ResponseField>

<ResponseField name="feedback_summary" type="object" required>
  Summary of feedback data used
  
  <Expandable title="feedback summary properties">
    <ResponseField name="total_feedback_count" type="integer" required>
      Total feedback entries processed
    </ResponseField>
    
    <ResponseField name="feedback_types" type="object" required>
      Breakdown by feedback type
    </ResponseField>
    
    <ResponseField name="quality_distribution" type="object" required>
      Distribution of feedback quality scores
    </ResponseField>
    
    <ResponseField name="common_patterns" type="array" required>
      Common patterns identified in feedback
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="generated_at" type="datetime" required>
  Summary generation timestamp
</ResponseField>
