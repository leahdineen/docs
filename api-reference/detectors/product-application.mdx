title: 'product_application'

## Overview

The `product_application` detector identifies and analyzes how products are being used, applied, or demonstrated in video content. It verifies that product application is visible and properly demonstrated when required by brand guidelines.

## Configuration

<ParamField body="config.require_application" type="boolean">
  Whether product application demonstration is required for the content to pass
</ParamField>

## Example Configuration

```json
{
  "detector_name": "product_application",
  "config": {
    "require_application": true
  }
}
```

## Interpreting Results

The product_application detector analyzes product usage demonstrations and returns:

- **Application Detected** - Boolean indicating whether product application is visible
- **Application Segments** - Time ranges showing product usage with quality assessments
- **Product Type** - Category of product being applied (skincare, makeup, hair, etc.)
- **Application Method** - How the product is applied (facial_application, brush_application, etc.)
- **Technique Quality** - Assessment of application skill (proper, professional, adequate, poor)
- **Demonstration Quality** - Score from 0.0 to 1.0 rating the overall quality of the product demonstration
