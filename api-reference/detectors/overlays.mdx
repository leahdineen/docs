## Overview

The `overlays` detector identifies text overlays, graphics, stickers, watermarks, and other visual elements that appear on top of the main video content. It uses OpenAI Vision API to analyze and categorize different types of overlays.

## Configuration

<ParamField body="config.categories" type="array">
  Categories of overlays to detect (e.g., ["text_overlay", "sticker", "watermark", "graphic"])
</ParamField>

<ParamField body="config.exclude_from_overlays" type="array">
  Elements to exclude from overlay detection (e.g., ["subtitles", "timestamps"])
</ParamField>

## Example Configuration

```json
{
  "detector_name": "overlays",
  "config": {
    "categories": ["text_overlay", "sticker", "graphic"],
    "exclude_from_overlays": ["subtitles", "timestamps"]
  }
}
```

## Overlay Categories

### Text Overlays
- **Product Information** - Names, prices, descriptions
- **Call-to-Action** - "Swipe up", "Link in bio", "Shop now"
- **Captions** - User-added text descriptions
- **Quotes** - Highlighted text or sayings

### Graphics
- **Brand Logos** - Company or product logos
- **Icons** - Functional or decorative icons
- **Borders/Frames** - Decorative frame elements
- **Shapes** - Geometric overlays or highlights

### Stickers
- **Emoji** - Animated or static emoji stickers
- **Reactions** - Facial expressions or emotions
- **Decorative** - Fun or themed sticker elements
- **Interactive** - Polls, questions, or engagement stickers

### Watermarks
- **Platform Watermarks** - Social media platform branding
- **Creator Watermarks** - Personal or brand watermarks
- **Copyright** - Ownership or attribution marks
- **Security** - Anti-piracy or protection overlays

## Position and Size Analysis

### Positions
- **top_left**, **top_center**, **top_right**
- **middle_left**, **center**, **middle_right**  
- **bottom_left**, **bottom_center**, **bottom_right**

### Sizes
- **small** - Minimal screen coverage (<10%)
- **medium** - Moderate coverage (10-30%)
- **large** - Significant coverage (>30%)

## Use Cases

- **Content Authenticity** - Detecting added graphics vs. organic content
- **Brand Compliance** - Ensuring proper logo placement and sizing
- **Platform Requirements** - Meeting overlay guidelines for different platforms
- **Quality Control** - Managing overlay density and placement

## Detector Type

**AI_VISION** - Uses OpenAI Vision API for visual element detection and categorization
